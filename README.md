  <h2>Sobre</h2>
  <p>Somos um e-commerce voltado para a venda de produtos naturais ou de materiais recicláveis. Além dos nossos produtos de baixo custo, também oferemos produtos gratuitos com o intuito de garantir uma vida melhor no futuro. Nossas iniciativas foram criadas para combater as mudanças climáticas e consequentemente reduzir a desigualdade dentro do país, visando alcançar o crescimento de renda da população mais pobre do Brasil.</p>
